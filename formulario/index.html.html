Perguntas do ITEM 2:

  1. Na tag input com atributo type="text", para que serve o atributo "placeholder"?
  2. Na tag input com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?
  3. Na tag input para que serve o atributo "value"?
  4. Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?
  5. Para que serve a tag label? Para que serve o atributo "for" nessa tag?
  6. Na tag option para que serve o atributo "selected"?
  7. Nas tags de formulário para que serve o atributo "required"?
  8. Para que serve a tag textarea?
  9. Na tag form para que serve o atributo "action"?
  10. Na tag form para que serve o atributo "method"?
  11. Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag form?
  12. Qual valor deve ser informado no atributo "method" na tag form em um formulário que será usado para fazer upload de arquivos?
  13. O que as tags fieldset e legend produzem?

Respostas do ITEM 2:

  1. O atriuto placeholder serve para deixar um expaço de texto reservado
  2. O atributo checked serve para deixar uma caixa de seleção pré-selecionada.
  3. O atributo value e a resposta ou escolha, o value pode ser a resposta entre escolher um carro ou uma bicicleta.
  4. O atributo que deve ser usado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez e o "name".
  5. A tag label define um rótulo para vários elementos ja o atibuto for Especifica o id do elemento do formulário ao qual o rótulo deve estar vinculado.
  6. O atributo selected Especifica que uma opção deve ser pré-selecionada quando a página for carregada
  7. O requiredatributo é um atributo booleano. Quando presente, especifica que um campo de entrada deve ser preenchido antes do envio do formulário.
  8. textarea tag define um controle de entrada de texto multilinha.
  9. O action atributo especifica para onde enviar os dados do formulário quando um formulário é enviado.
  10. O atributo method especifica o método HTTP a ser usado ao enviar os dados do formulário.
  11. o valor do atributo method deve ser o POST.
  12. o valor informado no atributo method para uploud de arquivos é o POST pois e mmais seguro e confiavel e seguro.
  13. A tag fieldsettag é usada para agrupar elementos relacionados em um formulário e a tag desenha uma caixa ao redor dos elementos relacionados,A tag legend define uma legenda para o elemento fieldset.
